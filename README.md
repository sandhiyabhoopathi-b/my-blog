# Django Blog Project

A simple Django-based blogging platform where users can create, view, update, and delete blog posts. This project demonstrates CRUD operations, Django models, templates, and static files usage.

## **Features**

- Create new blog posts with a title and content.
- View all blog posts in a responsive layout.
- Update and delete existing posts.
- Navigation bar and footer with social media links.
- Uses Django templates for a clean UI.
- Posts are stored in a SQLite database (can be changed to MySQL/PostgreSQL).
